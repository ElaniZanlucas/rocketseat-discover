## Basic Font Properties

* font-family
* font-weight
* font-style
* font-size

-------------------------------------------------------

## Font Family

* Tipo de fonte de um elemento
* Lista de fontes e odrem de prioridade
* inclui *fallback* font (escape)

```css
p {
  font-family: "Times New Roman", Times, serif;
}
```

  - serif (fonte com dobrinhas nas letras)
  - sans (fonte sem dobrinhas nas letras)

-------------------------------------------------------

## Font Weight

* Peso da fonte

```css
p {
  font-weight: bold;
}
```

-------------------------------------------------------

## Font Style

* O estilo da fonte

```css
p {
  font-style: italic;
}
```
-------------------------------------------------------

## Font Size

* O tamanho da fonte

```css
p {
  font-size: 30px;
}
```
